# 🤖 AI Student Assistant

An AI-powered web application designed to help students understand academic concepts through detailed explanations, visual learning resources, and related educational videos.

The AI Student Assistant allows users to ask questions on any topic and receive structured explanations generated using Artificial Intelligence.

---

## 🚀 Project Overview

AI Student Assistant is a full-stack web application that integrates modern web technologies with Artificial Intelligence to create an interactive learning environment. The system accepts student queries and generates detailed explanations along with related learning media to improve understanding.

This project demonstrates frontend development, backend API integration, cloud deployment concepts, and secure handling of API credentials.

---

## ✨ Features

- ✅ Ask academic questions interactively
- ✅ AI-generated detailed explanations
- ✅ Concept-based learning assistance
- ✅ Related educational video suggestions
- ✅ Clean and student-friendly interface
- ✅ Backend API integration
- ✅ Secure API key management using environment variables
- ✅ Responsive web design

---

## 🧠 How It Works

1. User enters a question or problem.
2. The frontend sends the request to the backend server.
3. The Node.js server securely communicates with the Gemini AI API.
4. AI generates an explanation.
5. The response is displayed along with related learning resources.
